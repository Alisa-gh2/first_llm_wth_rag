# F1r5t_LLM_W1th_R4G
Это мой первый масштабный проект в ML, где реализован бэк на FastAPI для работы на базе LLM с RAG-пайплайном для ответов на вопросы по учебным документам. 

# Корпус документов и постановка задачи
В качестве основной предметной области будут учебные тексты, то есть файлы лекций, конспектов и прочего с целью суммаризации, подготовки блиц-опросов и других методов, позволяющих оптимизировать и усовершенствовать процесс изучения и усвоения материалов.

Для тестовой выборки текстов по темам Linux, Windows и Сети были сформулированы следующие типовые пользовательские запросы, условно разбитые на 2 группы.

1. Информационные вопросы: 
Что такое Linux?
Что такое Docker?
Чем отличается Windows от Linux?
Какие команды в Linux нужны для манипуляций файлами?
Какие пакетные менеджеры существуют в Linux?
Когда появился Linux?
Кто такой root и чем он отличается от администратора?
Что такое Active Directory?
Чем отличается рабочая группа от домена?
Какие роли может выполнять Windows Server?
Что такое контроллер домена и зачем он нужен?
Какие объекты входят в Active Directory?

2. Уточняющие вопросы:
Какая структура диалога Kerberos между клиентом, KDC и сервером?
Какие шаги выполняет клиент при аутентификации по NTLMv2?
Сколько символов максимально может содержать пароль в NTLM и какие алгоритмы хэширования используются?
В чем особенности дистрибутивов Linux Kali и Ubuntu?
Что такое Debian и почему он считается достаточно надежным по сравнению с другими?
Что такое RWX в Linux?

Тестовая выборка текстов находится в папке /docs.
